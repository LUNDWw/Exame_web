# Ótica da Família

Site para agendamento de exames oftalmológicos com formulário integrado ao WhatsApp.

## Funcionalidades
- Formulário com máscara para CPF e telefone
- Envio automático dos dados para o WhatsApp da loja
- Design responsivo com Bootstrap 5
- Cards interativos com efeito hover

## Tecnologias
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript
- jQuery + Máscaras

## Como usar
1. Preencha nome, telefone e CPF
2. Clique em "Marcar Exame"
3. A mensagem será aberta no WhatsApp para confirmação
